# Навык: landing-copy

Этот навык предназначен для создания точных копий посадочных страниц на основе существующих компонентов, адаптируя их под конкретные SEO-запросы.

## Принципы работы
1. **Анализ структуры**: Навык анализирует `src/pages/index.astro` для определения набора необходимых компонентов.
2. **Переиспользование компонентов**: Все компоненты в `src/components/Shared/` должны поддерживать передачу данных через `props`.
3. **SEO Оптимизация**: Для генерации контента используются данные из папки `seo/` и навык `seo.md`.
4. **Моковые данные**: Каждая новая страница должна содержать уникальные моковые данные, соответствующие тематике.
5. **Сохранение структуры**: HTML-структура компонентов остается неизменной, меняется только текстовое наполнение и изображения (через props).

## Алгоритм создания новой страницы
1. Выбрать целевой SEO запрос.
2. Создать новый файл в `src/pages/[slug].astro`.
3. Сформировать объект данных (моки) на основе SEO ключей.
4. Импортировать компоненты из `src/components/Shared/`.
5. Передать моковые данные в компоненты.

## Требования к компонентам
- Все текстовые блоки должны быть вынесены в `Astro.props`.
- Должны быть определены значения по умолчанию (default values), соответствующие текущему контенту `index.astro`.
