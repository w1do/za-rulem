# План разработки контента для сайта авто тех помощью на дороге в тюмени
Данный сайт должен помогать найти услугу связанную с помощью на дорогах в городе тюмени И области

## Главные ключевые запросы (сем.ядра)
Каждая страница, это каждая страница посадочная под запрос смотреть папку `seo/`

Этот план описывает стратегию наполнения сайта экспертным контентом на русском языке с учетом SEO и фокуса на бизнес-результат.

## План проекта
Построить просто лёгкий сайт, который обойдет конкурентов по seo, будет наполняемым под каждый кластер, автоматически закрывать интент поиска услуги

> Очень важен кластер запросов, не просто `помощь на дорогах` а охватить данный кластер подзапросами `прикурить авто`, `дотащить до сервиса`, `замена аккумулятора с выездом`
!
